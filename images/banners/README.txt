Put the six section banner images in this folder, named exactly:

    standings.jpg   (the aerial stadium)
    scorers.jpg     (the ball in the net)
    bracket.jpg     (the gold trophy)
    players.jpg     (the player silhouette)
    stats.jpg       (the overhead pitch)
    ai.jpg          (the data / network texture)

.png or .jpeg also work. Each section page picks its image up automatically.
If an image is missing, that section shows a plain dark banner instead.
