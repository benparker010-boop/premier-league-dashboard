Google Calendar sync

Pull events from a Google Calendar in as jobs. Each event becomes a job (event title → job name, location → location, start/end → setup/collection dates). Re-syncing updates existing jobs instead of duplicating them.

In Google Calendar: Settings → Settings for my calendars → pick your jobs calendar → Integrate calendar → copy the “Secret address in iCal format.” Treat it like a password — anyone with this link can read that calendar.
